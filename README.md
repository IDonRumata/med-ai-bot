# MedAI Bot — Личный медицинский ИИ-ассистент

Приватный Telegram-бот для отслеживания здоровья с использованием OpenAI GPT-4o.

## Возможности

- **Загрузка анализов** (PDF/фото) — извлечение показателей, сохранение, краткая сводка
- **Логирование жалоб** (голос/текст) — расшифровка, контекстный анализ, рекомендации
- **Тренд-анализ** — графики динамики показателей с оценкой трендов
- **Проактивные напоминания** — follow-up по жалобам, напоминания о чекапах
- **Экспорт для врача** — структурированный отчёт с анамнезом и графиками

## Безопасность

- Whitelist по `telegram_id` — бот реагирует только на одного пользователя
- Все секреты в `.env` — не в коде
- Шифрование чувствительных данных в БД (Fernet)
- Анонимизация перед отправкой в OpenAI (удаление ФИО, телефонов, email)
- Docker-изоляция, закрытые порты PostgreSQL

## Оптимизация расходов

- **GPT-4o-mini** для парсинга анализов и простого тренд-анализа
- **GPT-4o** только для сложного медицинского анализа (симптомы, Vision)
- Локальное извлечение текста из PDF (PyMuPDF) — без API
- Локальная оценка severity — без API

## Быстрый старт

```bash
# 1. Клонировать
git clone https://github.com/IDonRumata/med-ai-bot.git
cd med-ai-bot

# 2. Настроить
cp .env.example .env
# Заполнить .env реальными значениями

# 3. Сгенерировать ключ шифрования
python -c "from cryptography.fernet import Fernet; print(Fernet.generate_key().decode())"

# 4. Запустить
docker compose up -d --build

# 5. Проверить логи
docker compose logs -f bot
```

## Стек

- Python 3.11, aiogram 3.x
- PostgreSQL 16 + SQLAlchemy + Alembic
- OpenAI GPT-4o / GPT-4o-mini / Whisper
- APScheduler, PyMuPDF, matplotlib
- Docker Compose

## Команды бота

| Команда | Описание |
|---------|----------|
| `/start` | Приветствие и справка |
| `/profile` | Настройка профиля (возраст, пол, хронические) |
| `/metrics` | Список всех сохранённых показателей |
| `/trend <название>` | График и анализ динамики показателя |
| `/export` | Сводный отчёт для врача |
| `/help` | Справка |
