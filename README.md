**Med AI Bot** — персональный медицинский AI-ассистент в Telegram. Отвечает на вопросы про симптомы, лекарства, анализы; ведёт дневник здоровья; напоминает о приёме лекарств и осмотрах.

> ⚠️ **Дисклеймер:** Информационный инструмент. Не заменяет врача. При подозрении на серьёзное заболевание → срочно к врачу.

**Целевая аудитория:** Люди, управляющие хроническими заболеваниями (диабет, гипертония, аллергии)  
**Регулирование:** Compliance с медицинскими стандартами (HIPAA-like для приватности)

---

## ✨ Ключевые возможности

| Функция | Описание |
|---------|---------|
| 💊 **Вопросы про лекарства** | Информация о препаратах, побочные эффекты, взаимодействия |
| 📋 **Интерпретация анализов** | Что означают значения крови, мочи (общая информация) |
| 📅 **Дневник здоровья** | Логирование симптомов, давления, веса, сна |
| 🔔 **Напоминания** | О приёме лекарств, осмотрах, анализах |
| 🏥 **Список врачей** | Сохраняй контакты своих докторов и даты осмотров |
| 📊 **Статистика** | Тренды (давление за месяц, вес за полгода) |
| 🆘 **SOS функция** | Быстрый доступ к скорой помощи в твоём городе |

## 📊 Примеры

### Вопрос про лекарство

```
Юзер: Какие побочки у Ибупрофена?

Бот: 💊 Ибупрофен (НПВС)
     Общие побочные эффекты:
     • Боль в животе, изжога
     • Головокружение
     • Аллергические реакции (редко)
     
     ⚠️ Осторожно при:
     • Язвах желудка (риск осложнений)
     • Заболеваниях почек
     • Беременности (2-3 триместр)
     
     Взаимодействия:
     Не принимай с антикоагулянтами.
     
     💬 Если волнуешься → спроси у своего врача
```

### Логирование симптомов

```
Юзер: /log
      Симптом: головная боль (сильная, 8/10)
      Давление: 140/90
      Время: сейчас

Бот: ✅ Записано!
     Это выше твоего обычного давления (120/80).
     
     Советы:
     • Отдохни 30 минут в темноте
     • Выпей воды
     • Если не пройдёт → примени твой обычный препарат
     
     Данные сохранены в дневнике (можешь посмотреть /history)
```

### Статистика

```
Юзер: /stats

Бот: 📊 Твоя статистика (30 дней)
     
     Давление: Avg 130/85 → Trend: стабильно
     Вес: 75kg → 74kg (↓ 1kg)
     Сон: Avg 7.5h (норма: 7-9h)
     
     Симптомы (топ 3):
     1. Усталость (5 раз)
     2. Головная боль (3 раза)
     3. Бессонница (2 раза)
     
     💡 Совет: Усталость коррелирует со стрессом.
               Попробуй медитацию 10 мин/день.
```
